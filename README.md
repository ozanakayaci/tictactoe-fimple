## Getting Started

### Prerequisites

- npm
  ```sh
  npm i nstall npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ozanakayaci/tictactoe-fimple.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Runs the app in the development mode.

   ```sh
   npm start
   ```
